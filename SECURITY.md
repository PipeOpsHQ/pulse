# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in Pulse, please report it privately to the maintainers at security@pipeops.io. We take all security reports seriously and will work with you to address the issue promptly.
