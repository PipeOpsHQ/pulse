# Contributing to Pulse

Thank you for your interest in contributing to Pulse! We welcome contributions from the community.

## How to Contribute

1. **Fork the Repository**: Create a fork of this repository to your own GitHub account.
2. **Clone the Repository**: Clone your fork to your local machine.
3. **Create a Branch**: Create a new branch for your changes.
4. **Make Changes**: Implement your changes and ensure they follow our coding style.
5. **Test Your Changes**: Run tests to verify your changes.
6. **Submit a Pull Request**: Submit a pull request to the main repository.

## Coding Standards

- **Go**: Follow standard Go formatting (`gofmt`).
- **Frontend**: Use Tailwind CSS for styling and follow Svelte best practices.
- **Commit Messages**: Use descriptive commit messages.

## Community

Join our community discord or forum to discuss ideas and get help.
